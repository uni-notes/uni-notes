## Program 1

**Add two 2-digit hexadecimal numbers** (method 1)

```assembly
DATA SEGMENT

ORG 1000H
NUM1 DB 89H
NUM2 DB 7CH
RES DW ?

DATA ENDS
CODE SEGMENT
ASSUME CS: CODE, DS: DATA

START:
MOV AX, DATA
MOV DS, AX
MOV AH, 0
MOV AL, NUM1
ADD AL, NUM2
ADC AH, 0
MOV RES, AX
INT 3

CODE ENDS
END START
```

## Program 3

**Write a program to add two 2-digit decimal numbers available in memory and store the result in memory.**

$$
89+78 = 167
$$

```assembly
DATA SEGMENT
ORG 1000H
NUM1 DB 89H
NUM2 DB 78H
RES DW ?
DATA ENDS

CODE SEGMENT
ASSUME CS: CODE, DS: DATA

START:

MOV AX, DATA
MOV DS, AX
MOV AH, 0
MOV AL, NUM1
ADD AL, NUM2
DAA
ADC AH, 0
MOV RES, AX
INT 3

CODE ENDS
END START
```

## `DAA`

DAA corrects the result of a previous addition of two valid packed decimal operands (note that this result must be in AL). This instruction changes the content of AL so that it will contain a pair of valid packed decimal digits.
